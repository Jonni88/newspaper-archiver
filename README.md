# Архив газет Олёкминска

Приложение для автоматической обработки архивных PDF газет и формирования базы событий для рубрики "Этот день в истории Олёкминска".

## Возможности

- 📥 Импорт PDF файлов (текстовых и сканов)
- 🔍 Автоматическое распознавание текста (OCR для сканов)
- 📋 Извлечение событий с датами, местами, участниками
- 📅 Формирование рубрики "Этот день в истории"
- 💾 Экспорт в CSV/JSON/TXT

## Установка Tesseract OCR

### Windows
1. Скачать установщик: https://github.com/UB-Mannheim/tesseract/releases
2. Установить с опцией "Add to PATH"
3. Проверить: `tesseract --version`

## Сборка .exe (GitHub Actions)

Сборка происходит автоматически на GitHub при пуше:
1. Запушь изменения: `git push origin master`
2. Зайди на вкладку **Actions** в репозитории
3. Скачай готовый архив с артефактами

## Сборка .exe (локально)

### 1. Установить Tesseract
См. раздел выше.

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

**Примечание:** Для OCR требуется PaddlePaddle. На Windows:
```bash
pip install paddlepaddle-gpu
```

## Запуск

```bash
python main.py
```

## Сборка .exe

```bash
python build_exe.py
```

Или вручную:
```bash
pyinstaller --clean build_exe.spec
```

Результат будет в папке `dist/`.

## Структура проекта

```
newspaper-archiver/
├── app/              # UI (PySide6)
├── core/             # Обработка PDF, OCR, извлечение событий
├── db/               # Модели и репозитории SQLite
├── assets/           # Ресурсы
├── main.py           # Точка входа
├── requirements.txt  # Зависимости
└── build_exe.py      # Скрипт сборки
```

## База данных

SQLite база содержит таблицы:
- `issues` — выпуски газет
- `pages` — страницы с текстом
- `events` — извлеченные события
- `jobs` — очередь задач обработки

## Использование

1. Откройте приложение
2. Нажмите "Выбрать PDF" или "Выбрать папку"
3. Дождитесь обработки (прогресс в таблице задач)
4. Перейдите во вкладку "События" для просмотра
5. Используйте "Этот день" для формирования рубрики
